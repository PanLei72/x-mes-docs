= 操作

操作是定义工艺路径的构建块。每个操作定义了在工艺路径的每个步骤中必须执行的活动。

== 相关对象
操作与以下对象相关:

* 工作中心
* 工序
* 作业指导书


== 数据存储
有关操作的信息存储在以下数据库表中:

* OPERATION
* ROUTE_STEP

== 操作浏览页面
由以下部分组成:

* *过滤器*

定义过滤条件，默认提供操作名称、描述过滤条件。也可以自定义配置过滤器。

* *功能按钮*

创建、编辑、删除、导出操作对象。

分页查询，修改每页显示行数。

* *数据列表*

显示操作对象所有属性。
