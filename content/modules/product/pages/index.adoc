= 产品

`产品` 是指正在生产的物品或项目，和工单关联。

== 相关对象
产品与以下对象相关:

* 工单


== 数据存储
有关产品的信息存储在以下数据库表中:

* PRODUCT

== 产品浏览页面
由以下部分组成:

* *过滤器*

定义过滤条件，默认提供产品名称、描述过滤条件。也可以自定义配置过滤器。

* *功能按钮*

创建、编辑、删除、导出产品对象。

分页查询，修改每页显示行数。

* *数据列表*

显示产品对象所有属性。
