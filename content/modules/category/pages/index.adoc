= 实体类别

`实体类别` 用于定义其他对象的类别属性。

== 相关对象
实体类别与所有存在类别属性的对象相关。


== 数据存储
有关实体类别的信息存储在以下数据库表中:

* CATEGORY

== 实体类别浏览页面
由以下部分组成:

* *过滤器*

定义过滤条件，默认提供实体、类别名称过滤条件。也可以自定义配置过滤器。

* *功能按钮*

创建、编辑、删除、导出实体类别对象。

分页查询，修改每页显示行数。

* *数据列表*

显示实体类别对象所有属性。
