= 设备

设备对象是分配给工作中心处理对象的资源对象。

== 相关对象
设备与以下对象相关:

* 工作中心
* 位置
// * 作业指导书
* 工作日历


== 数据存储
有关设备的信息存储在以下数据库表中:

* EQUIPMENT
* WORK_CENTER_EQUIPMENT

== 设备浏览页面
由以下部分组成:

* *过滤器*

定义过滤条件，默认提供设备名称、描述过滤条件。也可以自定义配置过滤器。

* *功能按钮*

创建、编辑、删除、导出设备对象。

分页查询，修改每页显示行数。

* *数据列表*

显示设备对象所有属性。
